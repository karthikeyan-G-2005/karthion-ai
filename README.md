# 🤖 Karthion AI

Karthion AI is a web-based AI chatbot that allows users to interact with an intelligent assistant in real time. The application uses the OpenRouter API to generate AI-powered responses and provides a clean and interactive chat interface.

This project is built using Node.js, Express, HTML, CSS, and JavaScript and is designed as a simple conversational AI platform for learning and experimentation.

---

## 🚀 Features

* 💬 AI-powered chatbot responses
* 🧠 Chat history management
* 🎤 Voice input support
* 📋 Copy AI responses
* 📝 Markdown formatted responses
* 🔍 Search chat conversations
* 📱 Responsive user interface
* ⚡ Fast API communication

---

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/karthikeyan-G-2005/karthion-ai.git
```

2. Navigate to the project folder

```
cd karthion-ai
```

3. Install dependencies

```
npm install
```

4. Create a `.env` file

```
OPENROUTER_API_KEY=your_api_key_here
```

5. Start the server

```
npm start
```

6. Open in browser

```
http://localhost:3000
```
